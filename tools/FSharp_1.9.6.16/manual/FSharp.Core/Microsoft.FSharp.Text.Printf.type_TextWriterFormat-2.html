[Home] Type Microsoft.FSharp.Text.Printf.TextWriterFormat


Represents a statically-analyzed format associated with writing to a System.IO.TextWriter. The first type parameter indicates the arguments of the format operation and the last the overall return type. Note: an abbreviation for Format<'T,TextWriter,unit,'Result>

Full Type Signature

[Home] Type Microsoft.FSharp.Collections.Tagged.HashSet


Mutable hash sets based on F# structural "hash" and (=) functions. Implemented via a hash table and/or Dictionary. Mutable hash sets where a constraint tag tracks information about the hash/equality functions used for the hashing. When the tag is Tags.StructuralHash this is identical to HashSet.
